# stack-calc
Это простой стековый калькулятор, который преобразует инфиксное выражение в постфиксное (обратная польская нотация, RPN) и затем вычисляет результат. Реализовано на языке C, с использованием двух стеков: для чисел и для операторов.

Что делает программа:
1.Принимает одно аргумент-строку — инфиксное выражение (в кавычках при запуске из командной строки).
2.Проверяет корректность символов и баланс скобок.
3.Преобразует инфиксное выражение в постфикс (функция infixToPostfix).
4.Вычисляет постфиксное выражение (функция evaluatePostfix) и выводит результат.

Пример запуска:
./cmdcalc "3+4"
# или с пробелами
./cmdcalc "5 * ( 1 + 2 ) - 3"
# пример с десятичными числами
./cmdcalc "3.5  * (2 + 1.5)"   # но лучше писать в обычной инфиксной форме: "3.5*(2+1.5)"
